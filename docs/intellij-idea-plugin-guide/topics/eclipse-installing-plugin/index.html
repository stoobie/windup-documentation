Procedure

  1. Launch Eclipse or CodeReady Studio.

  2. From the menu bar, select Help → Install New Software.

  3. Next to the Work with field, click Add.

  4. In the Name field, enter {ProductShortName}.

  5. Select all the JBoss Tools - MTA check boxes and click Next.

  6. Review the installation details and click Next.

  7. Accept the terms of the license agreement and click Finish.

  8. Restart Eclipse or CodeReady Studio.
